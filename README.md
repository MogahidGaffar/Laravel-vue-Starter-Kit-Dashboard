To get started, follow these commands:

```bash

git clone https://github.com/MogahidGaffar/Laravel-vue-Starter-Kit-Dashboard.git

# Change directory
cd Laravel-vue-Starter-Kit-Dashboard

# Install dependencies
composer install

# Install Node.js dependencies
npm install
